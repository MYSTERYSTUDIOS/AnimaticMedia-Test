# UE5 Sequencer Animation Recreation (AnimaticMedia Test)

## Overview
This project is a completed Unreal Engine test task focused on recreating an animated PNG sequence inside a cinematic scene using Sequencer.

## Features
- PNG sequence playback via Media Framework
- Material Instances based on master material
- Sequencer animation setup
- CineCamera configuration (16:9 DSLR)
- Accurate timing based on reference video
- Final rendering using Movie Render Queue

## Technical Details
- Engine: Unreal Engine 5.5.4
- Tools: Sequencer, Media Player, Media Texture
- Rendering: Movie Render Queue (8+2 preset workflow)

## What was done
- Imported static and animated PNG assets
- Created materials and material instances
- Set up media playback system
- Recreated animation timing from reference video
- Configured cinematic camera
- Built final sequence and rendered output

## How to Run
1. Download the project "UE_ArtistTestProject_GaleevDenis_Anim" from Releases
2. Extract the archive
3. Open the .uproject file
4. Open the main level
5. Open the Level Sequence in Sequencer
6. Play the sequence

## Notes
This project was completed as part of a test assignment and simulates a real production pipeline task.
